# GSQoderAI Workspace

This React TypeScript project was extracted and built from the `__QoderWork__.json` Zite export file.

## Tech Stack
- **Framework**: Vite + React 19 + TypeScript
- **Styling**: Tailwind CSS v3
- **Animations & Graphics**: Framer Motion, Recharts, Lucide React

## Setup & Run Instructions
1. Set the directory `/Users/mac/.gemini/antigravity-ide/scratch/qoder-work` as your active workspace.
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build the application for production:
   ```bash
   npm run build
   ```

The development server is currently running at [http://localhost:5173/](http://localhost:5173/).
